# CEDA — Cosmological Explanation Diagnostic Audit

CEDA is a conservation-honest diagnostic framework for auditing early-universe proposals and separating **genuine dynamics** from **descriptive bookkeeping**.  
It is **not** a cosmological model and does **not** propose an alternative to inflation or ΛCDM.

## Quickstart
1. Fill out a Model Card: 	emplates/model-card.md
2. Run tests (start with D1): 	ests/
3. Write a Diagnostic Report: 	emplates/diagnostic-report.md
4. Add the run under examples/<paper-id>/

## Repo Map
- paper/ — canonical PDF(s)
- 	emplates/ — model card + report templates
- 	ests/ — diagnostic tests (D1–D4)
- protocol/ — validity gate, verdict rules, no-rescue rule
- examples/ — completed audit runs
